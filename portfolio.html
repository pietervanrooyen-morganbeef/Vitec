WeighTrackPro

WeighTrackPro, a comprehensive C# and SQL Server-based application designed for weighing and data management in industries handling bulk materials. The system facilitates real-time weighing data capture via RS232 port, manages company parameters including logo and contact details, and allows database configuration through a dedicated form. User roles and permissions are central to its security, governing access to features like weighing operations, reporting (Excel, PDF, email), and system settings. Additionally, it incorporates scale parameter management with automatic service and verification reminders via email, alongside robust auditing and error handling to ensure data integrity and system stability, demonstrating its focus on efficiency and reliability.

Barcode Creator

Barcode Creator, a powerful and intuitive desktop application for Windows designed to streamline your product labeling workflow. Barcode Creator seamlessly integrates with your existing product database, allowing you to instantly generate and preview industry-standard CODE_128 barcodes for any item in your inventory. Simply select a product from the dropdown list, specify the number of labels you need, and print high-quality barcodes complete with product descriptions and codes. Built with C# and powered by the robust ZXing.Net engine, Barcode Creator offers a reliable and efficient solution for all your in-house barcode printing needs.

iSybotec

Sybotec Mobile / Web app software For the mining industry. The electronic inspection system is designed to replace the current manual, error-prone process with a streamlined and efficient digital solution. Key points of interest from the user specifications include the development of a comprehensive suite of applications featuring a mobile app for on-site inspections and a web app for back-office management. Field staff will use the mobile app to conduct offline visual and charging tests, record on-site repairs, and manage digital job cards that can be co-signed with mine representatives. The web application will serve as the central hub for management, allowing for the creation of user accounts with different privilege levels and the organization of inspection teams. A significant feature is the system's ability to manage detailed profiles for each cap lamp charging rack and client mine, including complex "proxy" arrangements for third-party contractors. Finally, a major selling point is the powerful report generation capability, which will create inspection and service reports, and provide valuable inspection statistics to identify recurring issues and offer insight into maintenance needs.

iTeloma

Teloma Service App iTeloma is a specialized data management application designed to facilitate and record inspections of rescue packs for mining clients. The system features a secure login screen that accommodates four distinct user types: an "Admin" for full oversight, "Desga" for handling repair reports, "ukraine" for general reporting, and a "Customer Mines" view. All user accounts, as well as the primary data for mines and individual rescue packs, are manually uploaded to the database by the software developer to ensure a controlled and accurate data environment. Each user accesses the system using a unique username and password. The core function of the application revolves around conducting two different types of inspections on rescue packs, which are uniquely identified by serial numbers: Visual Inspections: This is a straightforward pass-or-fail assessment. If an item fails the inspection, the user can select from a pre-loaded list of fault codes or descriptions corresponding to the issue. A key feature is the ability to take a photo of the fault, which is then automatically imported into the system. The application is designed to be scalable, allowing for new types of rescue packs with their own unique fault descriptions to be added in the future. Leak Tests: This is a simple pass-or-fail test to verify the integrity of the rescue pack.
